# Avatar Images

This directory contains avatar images for user profiles in the Epic Esports India platform.

## Default Avatar

The `default.jpg` file is used as a placeholder when users don't upload their own avatar.

## Credits

All custom avatars should be uploaded by users themselves. The default avatar is a simple placeholder designed to be neutral and inoffensive.
